⚡ Quick Start

1

Install AetherPost

pip install aetherpost
2

One-Time Setup

aetherpost init

Interactive setup generates campaign.yaml + creates profiles automatically

3

Deploy & Automate

aetherpost plan
aetherpost apply

Preview content, then deploy. Continuous automation begins instantly.

🤖 For Claude Code & AI Tools

Two setup approaches available: Interactive (guided) or Automated (pre-configured files)

📋 Interactive Setup (One-time configuration)

pip install aetherpost
aetherpost init

# Single interactive session handles everything:
# 1. Project details → campaign.yaml generation
# 2. Platform selection → automated profile creation
# 3. API authentication → continuous automation setup
# 4. Then: plan → apply → fully automated promotion

✅ One setup, full automation - One setup, lifetime automation
✅ Smart generation - Profiles, content, posting schedules
✅ Zero maintenance - Runs continuously after init

🚀 Auto-Detection Setup (Perfect for Claude Code)

Pre-create files and AetherPost auto-detects them, skipping all interactive prompts:

🔴 Required Files

campaign.yaml - Project configuration

name: "MyAwesomeApp"
concept: "Revolutionary productivity tool for developers"
description: "Transform your workflow with intelligent automation"
urls:
  main: "https://myapp.com"
  github: "https://github.com/user/myapp"
  docs: "https://docs.myapp.com"
platforms:
  - twitter
  - bluesky
content:
  style: friendly
  action: "Try it now!"
  language: en
🔴 Required for API Access

.env.aetherpost - API credentials

# ===== AI PROVIDERS (Choose one) =====
# OpenAI (Required for content generation)
OPENAI_API_KEY=sk-proj-your-openai-key-here

# Alternative AI Providers (Optional)
ANTHROPIC_API_KEY=your-anthropic-key
COHERE_API_KEY=your-cohere-key

# ===== SOCIAL MEDIA PLATFORMS (Choose any combination) =====
# Configure only the platforms you want to use

# Twitter
TWITTER_API_KEY=your-twitter-api-key
TWITTER_API_SECRET=your-twitter-api-secret
TWITTER_ACCESS_TOKEN=your-twitter-access-token
TWITTER_ACCESS_TOKEN_SECRET=your-twitter-access-token-secret
TWITTER_BEARER_TOKEN=your-bearer-token

# Bluesky
BLUESKY_HANDLE=username.bsky.social
BLUESKY_PASSWORD=your-app-password

# Instagram
INSTAGRAM_ACCESS_TOKEN=your-instagram-access-token
INSTAGRAM_BUSINESS_ACCOUNT_ID=your-business-account-id

# YouTube
YOUTUBE_CLIENT_ID=your-youtube-client-id
YOUTUBE_CLIENT_SECRET=your-youtube-client-secret
YOUTUBE_REFRESH_TOKEN=your-refresh-token

# Mastodon
MASTODON_ACCESS_TOKEN=your-mastodon-token
MASTODON_INSTANCE_URL=https://mastodon.social

# ===== AETHERPOST CONFIGURATION =====
AETHERPOST_LOG_LEVEL=INFO
AETHERPOST_CACHE_DIR=.aetherpost/cache
AETHERPOST_MAX_RETRIES=3
AETHERPOST_TIMEOUT=30
AETHERPOST_SAFE_MODE=false

# ===== CONTENT GENERATION =====
CONTENT_STYLE=friendly
CONTENT_LANGUAGE=en
CONTENT_MAX_LENGTH=280

# ===== SCHEDULING (Future Feature) =====
SCHEDULE_TIMEZONE=UTC
SCHEDULE_ENABLED=false

🌐 Platform Support & API Requirements

Choose any combination of platforms. Start small and expand as needed.

🐦

Twitter

160-character optimized tweets with engagement features

API Requirements:
TWITTER_API_KEY
TWITTER_API_SECRET
TWITTER_ACCESS_TOKEN
TWITTER_ACCESS_TOKEN_SECRET
Current Status (2025):
• Free tier: Read-only access only
• Basic ($100/month): 10K tweets read, 1.5K writes
• Pro ($5,000/month): 1M tweets read, 300K writes
• Enterprise: Custom pricing
💰 Requires paid Basic plan ($100/month minimum)
🦋

Bluesky

Decentralized social media with 256-character limit

API Requirements:
BLUESKY_HANDLE (username.bsky.social)
BLUESKY_PASSWORD (app password)
Current Status (2025):
• Completely free API access
• No rate limits on posting
• Open-source AT Protocol
• Growing user base (15M+ users)
✅ Free API access - Recommended
📸

Instagram

Visual content optimization and profile management

API Requirements:
INSTAGRAM_ACCESS_TOKEN
INSTAGRAM_BUSINESS_ACCOUNT_ID
Current Status (2025):
• Basic Display API: Free (profile only)
• Graph API: Free for basic features
• Requires Facebook Page connection
• Business account verification needed
⚠️ Business account + Facebook Page required
🐘

Mastodon

Decentralized open-source social network with community focus

API Requirements:
MASTODON_ACCESS_TOKEN
MASTODON_INSTANCE_URL
Current Status (2025):
• Completely free API access
• No corporate restrictions
• Choose your own instance
• Open-source community platform
✅ Free and open-source
📺

YouTube

Channel optimization and content strategy

API Requirements:
YOUTUBE_CLIENT_ID
YOUTUBE_CLIENT_SECRET
YOUTUBE_REFRESH_TOKEN (optional)
Current Status (2025):
• Free quota: 10,000 units/day
• Read operations: 1 unit each
• Write operations: 50-1600 units
• OAuth 2.0 authentication required
✅ Free quota (10,000 units/day)
