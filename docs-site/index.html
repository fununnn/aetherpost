⚡ Quick Start

1

Install AetherPost

pip install aetherpost
2

Initialize Project

aetherpost init main

Interactive setup with automatic profile generation

3

Generate Profiles

aetherpost profile generate

Create optimized profiles for all your platforms

4

Preview & Deploy

aetherpost plan
aetherpost apply

Preview AI-generated content and deploy across platforms

💡 Profile Generation Example

# campaign.yaml
name: "MyAwesomeApp"
description: "Revolutionary productivity tool"
urls:
  main: "https://myapp.com"
  github: "https://github.com/user/myapp"
  docs: "https://docs.myapp.com"

# Generated Twitter Profile:
Display Name: MyAwesomeApp 🚀
Bio: 👋 Building MyAwesomeApp! Revolutionary productivity tool for developers. Join me!
Website: https://myapp.com
Characters: 98/160

✅ Your app name, ✅ Your URLs, ✅ Your content, ❌ No AetherPost branding
