🤖 For Claude Code & AI Tools

Two setup approaches available: Interactive (guided) or Automated (pre-configured files)

📋 Interactive Setup (Recommended for first-time users)

pip install aetherpost
aetherpost init

# AetherPost will guide you through:
# 1. Project name and description input
# 2. Platform selection (Twitter, Bluesky, Instagram, etc.)
# 3. API key collection with direct links
# 4. Automatic profile generation
# 5. Campaign.yaml creation

✅ No files needed upfront - AetherPost creates everything for you
✅ Beginner-friendly - Step-by-step guidance
✅ API key validation - Tests connections immediately

🚀 Automated Setup (Perfect for Claude Code)

Pre-create files to skip all interactive prompts:

🔴 Required Files

campaign.yaml - Project configuration

name: "MyAwesomeApp"
concept: "Revolutionary productivity tool for developers"
description: "Transform your workflow with intelligent automation"
urls:
  main: "https://myapp.com"
  github: "https://github.com/user/myapp"
  docs: "https://docs.myapp.com"
platforms:
  - twitter
  - bluesky
content:
  style: friendly
  action: "Try it now!"
  language: en
🔴 Required for API Access

.env.aetherpost - API credentials

# ===== AI PROVIDERS (Choose one) =====
# OpenAI (Required for content generation)
OPENAI_API_KEY=sk-proj-your-openai-key-here                    # REQUIRED

# Alternative AI Providers (Optional)
ANTHROPIC_API_KEY=your-anthropic-key                           # OPTIONAL
COHERE_API_KEY=your-cohere-key                                 # OPTIONAL

# ===== SOCIAL MEDIA PLATFORMS =====
# Twitter (Requires $100/month Basic plan)
TWITTER_API_KEY=your-twitter-api-key                           # REQUIRED for Twitter
TWITTER_API_SECRET=your-twitter-api-secret                     # REQUIRED for Twitter
TWITTER_ACCESS_TOKEN=your-twitter-access-token                 # REQUIRED for Twitter
TWITTER_ACCESS_TOKEN_SECRET=your-twitter-access-token-secret   # REQUIRED for Twitter
TWITTER_BEARER_TOKEN=your-bearer-token                         # OPTIONAL (v2 API)

# Bluesky (Completely Free)
BLUESKY_HANDLE=username.bsky.social                            # REQUIRED for Bluesky
BLUESKY_PASSWORD=your-app-password                             # REQUIRED for Bluesky

# Instagram (Free with Business Account)
INSTAGRAM_ACCESS_TOKEN=your-instagram-access-token             # REQUIRED for Instagram
INSTAGRAM_BUSINESS_ACCOUNT_ID=your-business-account-id         # REQUIRED for Instagram

# YouTube (Free with OAuth)
YOUTUBE_CLIENT_ID=your-youtube-client-id                       # REQUIRED for YouTube
YOUTUBE_CLIENT_SECRET=your-youtube-client-secret               # REQUIRED for YouTube
YOUTUBE_REFRESH_TOKEN=your-refresh-token                       # OPTIONAL (auto-generated)

# Mastodon (Free - Open Source)
MASTODON_ACCESS_TOKEN=your-mastodon-token                      # REQUIRED for Mastodon
MASTODON_BASE_URL=https://mastodon.social                      # REQUIRED for Mastodon

# LinkedIn (Profile generation only - No API needed)
# GitHub (Profile generation only - No API needed)

# ===== AETHERPOST CONFIGURATION =====
AETHERPOST_LOG_LEVEL=INFO                                      # OPTIONAL (DEBUG, INFO, WARNING, ERROR)
AETHERPOST_CACHE_DIR=.aetherpost/cache                         # OPTIONAL (default: .aetherpost/cache)
AETHERPOST_MAX_RETRIES=3                                       # OPTIONAL (default: 3)
AETHERPOST_TIMEOUT=30                                          # OPTIONAL (default: 30 seconds)
AETHERPOST_SAFE_MODE=false                                     # OPTIONAL (default: false)

# ===== CONTENT GENERATION =====
CONTENT_STYLE=friendly                                         # OPTIONAL (friendly, professional, creative)
CONTENT_LANGUAGE=en                                            # OPTIONAL (en, ja, es, fr, de, etc.)
CONTENT_MAX_LENGTH=280                                         # OPTIONAL (platform-specific defaults)

# ===== SCHEDULING (Future Feature) =====
SCHEDULE_TIMEZONE=UTC                                          # OPTIONAL (default: UTC)
SCHEDULE_ENABLED=false                                         # OPTIONAL (default: false)

🌐 Platform Support & API Requirements

Choose any combination of platforms. Start small and expand as needed.

🐦

Twitter

160-character optimized tweets with engagement features

API Requirements:
TWITTER_API_KEY
TWITTER_API_SECRET
TWITTER_ACCESS_TOKEN
TWITTER_ACCESS_TOKEN_SECRET
Current Status (2025):
• Free tier: Read-only access only
• Basic ($100/month): 10K tweets read, 1.5K writes
• Pro ($5,000/month): 1M tweets read, 300K writes
• Enterprise: Custom pricing
💰 Requires paid Basic plan ($100/month minimum)
🦋

Bluesky

Decentralized social media with 256-character limit

API Requirements:
BLUESKY_HANDLE (username.bsky.social)
BLUESKY_PASSWORD (app password)
Current Status (2025):
• Completely free API access
• No rate limits on posting
• Open-source AT Protocol
• Growing user base (15M+ users)
✅ Free API access - Recommended
📸

Instagram

Visual content optimization and profile management

API Requirements:
INSTAGRAM_ACCESS_TOKEN
INSTAGRAM_BUSINESS_ACCOUNT_ID
Current Status (2025):
• Basic Display API: Free (profile only)
• Graph API: Free for basic features
• Requires Facebook Page connection
• Business account verification needed
⚠️ Business account + Facebook Page required
👩‍💻

GitHub

Developer-focused profile and README generation

Profile Generation Only:
No API keys needed for profile generation
✅ Free - Profile generation only
💼

LinkedIn

Professional networking profile optimization

Profile Generation Only:
No API keys needed for profile generation
📋 Profile generation only
📺

YouTube

Channel optimization and content strategy

API Requirements:
YOUTUBE_CLIENT_ID
YOUTUBE_CLIENT_SECRET
YOUTUBE_REFRESH_TOKEN (optional)
Current Status (2025):
• Free quota: 10,000 units/day
• Read operations: 1 unit each
• Write operations: 50-1600 units
• OAuth 2.0 authentication required
✅ Free quota (10,000 units/day)
